Coding Style
============

We try to comply with `PEP8`_ as much as possible with the only modification, that we allow a line length of up to 120
characters.

Our continuous integration server verifies all commits agains `PEP8`_ and with `pylint`_.

.. _PEP8: http://legacy.python.org/dev/peps/pep-0008/
.. _pylint: http://pylint.org/
